# AI for Teachers - GitHub Pages

Acest repository contine pagina workshopului pentru profesori.

## Continut
- `index.html` - pagina principala a workshopului
- `AI_for_Teachers.pptx` - prezentarea atelierului
- `anastasia_workshop.jpg` - imagine workshop

## Publicare pe GitHub Pages
1. Deschide repository-ul pe GitHub.
2. Intra la **Settings** > **Pages**.
3. La **Build and deployment**, selecteaza:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (root)
4. Salveaza configurarea.
5. Dupa 1-2 minute, pagina va fi disponibila la URL-ul afisat in sectiunea Pages.
